# When2Go
When2Go is an app for quickly finding the best time to start walking to the nearest Sydney light rail station. 

## Link
[when2go.netlify.app](https://when2go.netlify.app/) \
\
Recommended: Add it to your home screen as a shortcut!

## Features
- Cross platform (Android, iOS, web)
- Saves nearby stop and travel direction preferences
- Realtime tracking
- Auto refresh (30s) and manual refresh
